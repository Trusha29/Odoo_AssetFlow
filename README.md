# 🚀 AssetFlow

AssetFlow is an Enterprise Asset Management System developed for the **Odoo Hackathon**. It helps organizations efficiently manage assets, employees, bookings, maintenance, audits, and reports through a modern, user-friendly dashboard.

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- 📦 Asset Management
- 👥 Employee & Department Management
- 📅 Asset Booking
- 🛠 Maintenance Tracking
- 📋 Audit Management
- 📊 Reports & Analytics
- 🔔 Notifications
- 📱 Responsive UI

## 🛠 Tech Stack

**Frontend**
- React.js
- Vite
- Tailwind CSS
- React Router

**Backend**
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 🚀 Run Locally

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
